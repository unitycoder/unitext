# Modifiers Sample

Demonstrates UniText markup system and text modifiers.

## Supported Markup Tags

### Text Style
- `<b>Bold</b>`
- `<i>Italic</i>`
- `<u>Underline</u>`
- `<s>Strikethrough</s>`

### Color
- `<color=#FF0000>Hex color</color>`
- `<color=red>Named color</color>`

### Size
- `<size=24>Absolute size</size>`
- `<size=150%>Percentage</size>`
- `<size=+10>Relative</size>`

### Spacing
- `<cspace=5>Letter spacing</cspace>`
- `<line-height=1.5>Line height</line-height>`

### Links
- `<link=https://example.com>Clickable link</link>`

### Lists
- `- Bulleted item`
- `1. Numbered item`

### Inline Objects
- `<obj=iconName/>`

## Scene

Open `Modifiers.unity` to see all modifiers in action.

## Configuration

`ModRegisterConfig.asset` contains the modifier configuration for this sample.
