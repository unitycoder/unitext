# Emoji Sample

Demonstrates UniText emoji rendering capabilities.

## Features

- Standard emoji support (😀 🎉 ❤️)
- Country flags (🇺🇸 🇬🇧 🇯🇵)
- ZWJ sequences (👨‍👩‍👧‍👦)
- Skin tone modifiers (👍🏽)
- Emoji mixed with RTL text

## Scene

Open `Emoji.unity` to see emoji rendering in action.

## Notes

UniText uses a dedicated emoji font atlas for consistent cross-platform rendering.
