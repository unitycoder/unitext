# Layout Sample

Demonstrates UniText layout and alignment features.

## Features

- Horizontal alignment (Left, Center, Right)
- Vertical alignment (Top, Middle, Bottom)
- Word wrapping
- AutoSize (automatic font scaling)
- Line height and spacing control

## Scene

Open `Layout.unity` to explore layout options.

## Properties

- `HorizontalAlignment` — Left, Center, Right
- `VerticalAlignment` — Top, Middle, Bottom
- `WordWrap` — Enable/disable word wrapping
- `AutoSize` — Automatically scale font to fit container
- `MinFontSize` / `MaxFontSize` — AutoSize bounds
