# Font Examples

Demonstrates UniText multi-script and multi-font rendering.

## Fonts

- **Decorative:** Dancing Script, Lobster
- **Serif:** Noto Serif
- **Pixel:** Pixelify Sans
- **Monospace:** JetBrains Mono
- **Icons:** Font Awesome 7 Brands
- **Complex scripts:** Bengali, Devanagari, Tamil, Telugu, Khmer, Myanmar, Thai
- **CJK:** Noto Serif SC (Simplified Chinese)

## Scene

Open `FontExamples.unity` to see all fonts rendered side by side.
